## ğŸš€ Getting Started

### Prerequisites
- Node.js 16+
- MongoDB (local or Atlas)
- npm or yarn

### Installation

1. **Clone the repository**
```bash
cd mesmerizingmoments
```

2. **Setup Backend**
```bash
cd backend
npm install
```

3. **Setup Frontend**
```bash
cd frontend
npm install
```

### Running the Application

1. **Start MongoDB** (if using local)
```bash
mongod
```

2. **Start Backend** (from backend directory)
```bash
npm run dev
# or
npm start
```

3. **Start Frontend** (from frontend directory)
```bash
npm start
```

4. Open http://localhost:3000 in your browser
